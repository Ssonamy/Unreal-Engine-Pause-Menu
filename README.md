# Unreal Engine Pause Menu

## Что есть в проекте

- Пауза игры
- Меню настроек
- Работа с UI через UMG
- Сохранение базовых настроек

## Версия

Unreal Engine 5.x

## Как запустить

1. Скачать или клонировать репозиторий
2. Открыть файл `*.uproject`
3. Дождаться сборки шейдеров
4. Запустить проект

## Примечание

Проект сделан в учебных целях.
